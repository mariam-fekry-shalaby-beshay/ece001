    Java The Java™ Programming Language is a general-purpose, concurrent, strongly typed, class-based object-oriented language. It is normally compiled to the bytecode instruction set and binary format defined in the Java Virtual Machine Specification.
    Ruby Ruby is a dynamic, reflective, object oriented, general purpose programming language. Ruby is a pure Object-Oriented language developed by Yukihiro Matsumoto. In Ruby, everything is an entity except the blocks but there are also substitutes for it, i.e. procs and lambda. The purpose of Ruby 's development was to make it act as a sensible buffer between human programmers and the underlying computing machinery.
    HTML HTML is not a programming language; it is a markup language which defines your content structure. HTML consists of a collection of elements that you use to enclose, or cover, various sections of the material in order to make it appear in some way, or to behave in some way. The enclosing tags can hyperlink a word or picture to another location, can italicize words, can make the font bigger or smaller and so on .
    JavaScript JavaScript is one of the most popular programming languages in the world. Created in 1995, it’s gone a very long way since its humble beginnings.Being the first - and the only - scripting language that was supported natively by web browsers, it just stuck. it was mainly used for fancy animations and the marvel known at the time as DHTML. With the growing needs that the web platform demands, JavaScript had the responsibility to grow as well, to accommodate the needs of one of the most widely used ecosystems of the world.
    C C is a procedural programming language. It was initially developed by Dennis Ritchie in the year 1972. It was mainly developed as a system programming language to write an operating system. The main features of C language include low-level access to memory, a simple set of keywords, and clean style, these features make C language suitable for system programmings like an operating system or compiler development.
    C++ C++ is a general-purpose programming language which was designed to provide object-oriented paradigm as an enhancement of the C language. It is a compiled language and an imperative. C++ is a middle-level language that offers the advantage of low-level programming (drivers, kernels) and even higher-level applications (games, GUIs, desktop applications, etc.). Both C and C++ have the same basic syntax and code structure.
    C# C # is a general purpose programming language, modern and object-oriented, pronounced as "C Sharp." It was developed within the framework of the. NET initiative by Microsoft led by Anders Hejlsberg and his team, and was approved by the European Computer Manufacturers Association (ECMA) and the International Standards Organization ( ISO). C # is among Common Language Infrastructure Languages. C # is syntactically much like Java, and is simple for users who learn C , C++ or Java.
    Objective-C Objective-C is an object-oriented, general purpose programming language that applies Smalltalk-style messaging to the C programming language. This is Apple's primary programming language for the OS X and iOS operating systems and their respective APIs, Cocoa and Cocoa Touch.
